## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A web browser

### Setup

- Clone or download the project to your local machine

- Change to the project's root directory

- run `npm install`

- run `npm run start`

- Open the index file with your browser
```javascript
index.html
```
